Updating on iOS¶

How you update Battman on iOS depends on how you installed it.

Installed by TrollStore¶

  • Install the new version following the same method as the initial installation.

Package‑manager installs¶

  • Open your package manager (Cydia, Sileo, Zebra, etc.).
  • Refresh package sources.
  • When a new Battman version is available, use the Upgrade action.
