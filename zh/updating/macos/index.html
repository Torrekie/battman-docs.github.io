在 macOS 上更新 Battman¶

目前尚未发布 Battman 的原生 macOS 版本，因此本页主要说明未来预期的行为。

预期行为¶

  • macOS 版 Battman 可能会以签名应用包或通过包管理器的形式分发。
  • 更新方式大概率会遵循常见 macOS 应用模式（替换 /Applications 中的应用，或使用包管理器的更新命令）。

在 macOS 版本正式发布之前，本页无实际更新步骤可供执行；如需更新，请参见 更新 → iOS，该平台目前已得到正式支持。
